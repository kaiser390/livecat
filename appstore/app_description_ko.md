## 설명
아이폰을 OBS Studio용 프로 무선 카메라로 바꿔보세요.

OLiveCam은 로컬 Wi-Fi를 통해 고화질 H.264 영상을 OBS로 직접 전송합니다. 케이블도, 추가 플러그인도, 워터마크도 없습니다.

간단한 설정
1. 앱에서 PC의 IP 주소 입력
2. OBS에 미디어 소스 추가 (udp://@:9000)
3. 시작 탭 — 바로 방송

주요 기능
• 1080p / 720p H.264 하드웨어 인코딩
• 로컬 Wi-Fi 초저지연 전송
• 오디오 스트리밍 지원
• Windows, Mac, Linux OBS Studio 호환
• OBS 플러그인 불필요 — 표준 UDP/MPEG-TS
• PC에서 원격 줌 제어
• 연결 끊김 시 자동 재연결
• 배터리 및 발열 모니터링
• 스트리밍에 최적화된 가로 모드

이런 분께 추천
• 무선 카메라가 필요한 라이브 스트리머
• YouTube / Twitch 방송인
• 반려동물 캠, 홈 모니터링
• 브이로거 및 콘텐츠 크리에이터
• OBS 멀티캠 구성

프라이버시
• 로컬 네트워크만 사용 — 클라우드 없음, 데이터 수집 없음
• 계정 불필요
• 광고 없음, 추적 없음

아이폰, Wi-Fi, OBS. 이것만 있으면 됩니다.
