# LiveCatCam Project Context

## Overview
LiveCatCam - iOS cat tracking camera app. Detects cats via Vision framework, tracks with DockKit motor stands, streams video via SRT, sends metadata via WebSocket.

## Project Structure
- **iOS App**: `ios/LiveCatCam/` (Swift Package + Xcode project)
- **Server**: `server/` (Python - metadata_receiver.py, video_receiver.py, scene_analyzer.py, event_detector.py)
- **Xcode Project**: `ios/LiveCatCam/LiveCatCam.xcodeproj` (generated by xcodegen)

## iOS App Architecture (35 files, 7 modules)

### Models/
- `MetadataMessage.swift` - Server protocol messages (snake_case JSON), CatPosition, MotorPosition, CatInfo
- `CameraConfig.swift` - Server IP, cam ID, SRT port, metadata port 8081, resolution, fps
- `TrackingState.swift` - enum: idle, searching, tracking, lost
- `CatPose.swift` - 17 joint points from VNAnimalBodyPoseObservation
- `CatProfile.swift` - nana/toto profiles
- `ActivityEvent.swift` - climb/jump/run/interact/hunt with base scores
- `ServerCommand.swift` - set_mode, set_zoom, goto_position

### Utils/
- `KalmanFilter.swift` - 2D [x,y,vx,vy] state
- `CoordinateMapper.swift` - Vision coords -> DockKit angles (67deg HFOV, 41deg VFOV)
- `MovingAverage.swift` - EMA alpha=0.3
- `Logger.swift` - OSLog categories

### Network/
- `ServerConnection.swift` - WebSocket actor, registration, auto-reconnect
- `MetadataReporter.swift` - 10Hz timer
- `CommandReceiver.swift` - Dispatches ServerCommands
- `ConnectionMonitor.swift` - NWPathMonitor

### Tracking/
- `CatDetector.swift` - VNDetectAnimalBodyPoseRequest
- `CatTracker.swift` - Up to 2 cats, KalmanFilter, nearest-neighbor association
- `PoseClassifier.swift` - Rule-based classification
- `ActivityScorer.swift` - 5 components, 0-150 raw -> 0-100 normalized
- `TrackingStateMachine.swift` - State transitions with timeouts
- `MotorController.swift` - Protocol + DockKit impl (disabled with `#if ENABLE_DOCKKIT`)
- `MotorControllerMock.swift` - Simulates motor movement

### Capture/
- `CameraManager.swift` - AVCaptureSession, rear wide-angle
- `VideoEncoder.swift` - VTCompressionSession H.264, callback-based API
- `SRTStreamer.swift` - FIFO pipe -> ffmpeg-kit -> SRT caller
- `AudioCapture.swift` - Optional AAC audio

### App/
- `LiveCatCamApp.swift` - @main entry
- `AppState.swift` - @Observable @MainActor central coordinator, frame pipeline
- `ContentView.swift` - Root view

### UI/
- `StatusOverlayView.swift` - HUD overlay
- `DebugCameraView.swift` - Camera preview + Canvas overlay
- `ConfigurationView.swift` - Settings form

### Tests/
- `main.swift` - Executable test runner, 24/24 tests pass

### Support Files/
- `project.yml` - xcodegen spec
- `SupportFiles/Info.plist` - Camera/mic/network permissions, background modes
- `SupportFiles/LiveCatCam.entitlements` - DockKit entitlement (reset by xcodegen)
- `SupportFiles/Assets.xcassets/` - App icon placeholder

## Server Protocol
- WebSocket port 8081: registration `{"type": "register", "cam_id": "CAM-1"}`
- SRT ports 9000/9001 (listener mode)
- Metadata at 10Hz, stale timeout 3s
- Activity score 0-100, EMA alpha=0.3
- config.yaml for server settings
